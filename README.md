# 一卡通相片裁切工具
### Photo Cutter · by Fabrica

> 專為學校行政設計的學生大頭照批次處理工具。  
> 掃描相片表後自動裁切、辨識學號、匯出 ZIP，讓原本需要一整天的工作縮短到一小時內完成。

## 安裝

**系統需求：** Windows 10+ 或 macOS 12+，Python 3.10+

### 1. 安裝 Python 套件

```bash
pip install Pillow numpy PyJWT cryptography openpyxl
```

### 2. 下載程式

前往 [Releases 頁面](../../releases/latest) 下載 `photo_cutter.py`

### 3. 執行

```bash
python photo_cutter.py
```

### 選用功能（按需安裝）

| 功能 | 安裝指令 |
|---|---|
| OCR 自動學號辨識 | `pip install pytesseract` + 另裝 [Tesseract-OCR](https://github.com/UB-Mannheim/tesseract/wiki) |
| AI 去背白底 | `pip install rembg onnxruntime`（首次執行自動下載模型 ~170MB） |

## 使用教學

### 相片表模式（最常用）

```
步驟一  拖入掃描好的相片表圖片（JPG / PNG）
步驟二  程式自動偵測格線，照片立即顯示
        （OCR 在背景自動辨識學號，辨識完自動填入）
步驟三  逐張確認學號，點照片可放大，按 Enter 跳下一張
步驟四  按「儲存 ZIP」打包所有照片
```

> 💡 如果 Excel 有學生名冊，在步驟二前匯入，辨識時會同時顯示姓名方便核對。

### 批次照片模式（Scholar / Pro / Enterprise）

```
1. 選擇裝有學生照片的資料夾
2. 匯入 Excel 學生名單
3. 程式自動配對（檔名含學號自動比對）
4. 標示缺件學生（紅色）、未配對照片（黃色）
5. 手動補齊後批次匯出
```

## 常見問題

**Q：程式顯示「OCR OFFLINE」？**  
A：需要額外安裝 Tesseract-OCR 引擎。請參考[安裝說明](#選用功能按需安裝)。

**Q：虛線框沒有完全去除怎麼辦？**  
A：點照片旁的「✂ 精準裁切」按鈕，可以手動拖曳調整裁切位置。

**Q：Excel 匯入後學號對不上？**  
A：請確認 Excel 第一列為標題列，學號欄位名稱包含「學號」、「座號」或 `id` 等關鍵字。

**Q：處理到一半需要關閉電腦？**  
A：直接關閉即可。下次開啟同一張相片表，程式會自動詢問是否繼續上次進度。

**Q：可以同時處理多個班級？**  
A：目前每次處理一張相片表。可以依序開啟不同班級的圖片，進度各自獨立儲存。
